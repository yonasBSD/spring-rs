# Changelog

## 0.5.3

> Contains a **breaking** change to the code generated by the `sa-token` feature. Pair with `summer-sa-token >= 0.5.2`.

- **breaking**: under the `sa-token` feature, `#[sa_check_login]` / `#[sa_check_role]` / `#[sa_check_permission]` / `#[sa_check_roles_and]` / `#[sa_check_roles_or]` / `#[sa_check_permissions_and]` / `#[sa_check_permissions_or]` / `#[sa_ignore]` now emit calls to `summer_sa_token::sa_token_plugin_axum::StpUtil` instead of `summer_sa_token::StpUtil`. Requires `summer-sa-token >= 0.5.2`, which exposes the `sa_token_plugin_axum` namespace re-export.
